# 🎬 Popcorn Picks

Backend API for managing movies, reviews, and watchlists.

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- JWT Authentication

## Features
- User login
- Admin movie management
- Reviews & watchlist

## Run
- Open in IntelliJ
- Run Spring Boot application
- Test with Postman

## 🧠 What I Learned

This project was built as part of my backend development journey.  
Key things I focused on:

- Building a structured REST API with Spring Boot
- Implementing JWT authentication and role-based access (Admin/User)
- Writing unit and controller tests using JUnit and Mockito
- Handling validation and global exception handling
- Working with PostgreSQL and H2 for testing

This project helped me improve writing clean, testable backend code.
